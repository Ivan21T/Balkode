# Balkode IDE

> **A lightweight Integrated Development Environment (IDE) developed with PySide.**

Balkode IDE is a desktop code editor and development environment built using **Python** and **PySide**. The project focuses on providing a clean, fast, and user-friendly environment for writing, editing, and managing source code.

---

# Overview

Balkode IDE was developed to demonstrate desktop application development with Qt while providing essential tools for software development in a modern graphical interface.

---

# Technologies

* Python
* PySide (Qt for Python)
* Qt Framework

---

# Features

* Source code editor
* Syntax highlighting
* File explorer
* Open and save files
* Multi-file project support
* Modern graphical user interface
* Cross-platform desktop application
* Fast and lightweight design

---

# Goals

The goal of Balkode IDE is to provide a simple yet extensible development environment while demonstrating desktop application development using the Qt framework and PySide.
